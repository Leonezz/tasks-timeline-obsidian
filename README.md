# Tasks Timeline View for Obsidian

![](assets/1.png)

This is a view plugin to show and manage you task items in obsidian, it is a continuation of [obsidian-tasks-calendar-wrapper](https://github.com/Leonezz/obsidian-tasks-calendar-wrapper),
based on [tasks-timeline-components](https://github.com/Leonezz/tasks-timeline-components).
